\begin{abstract}\label{ch:abstract}
An abstract is a brief summarizing statement read by parties who are trying to decide whether or not to read the main document. Thus, it includes:
   \begin{itemize}
      \item a description of the problem 
      \item its relevance
      \item the method used to solve the problem
      \item a summary of the main results, and 
      \item main conclusions
   \end{itemize}
\end{abstract}
